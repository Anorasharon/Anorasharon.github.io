About Me

Hi, I'm Anora Sharon Tessie S — an Embedded Firmware Engineer passionate about designing intelligent hardware–software integrated systems.

I specialize in low-level system design, interrupt-driven firmware architecture, real-time embedded systems, and deterministic embedded development using MSP430, ESP32, and ARM platforms.

Core Technical Strengths:

  • Interrupt Service Routine (ISR) based firmware design
  • RTOS scheduling and inter-task communication
  • Peripheral interfacing (GPIO, UART, SPI, I2C, Timers)
  • Memory optimization & stack management
  • Modular architecture & state machine design

Alongside firmware development, I actively explore AI and Edge AI optimization, deploying TinyML-based inference systems on embedded platforms.
