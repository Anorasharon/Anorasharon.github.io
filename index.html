About Me

Hi, I'm Anora Sharon Tessie S — an Embedded Firmware Engineer passionate about designing intelligent hardware–software integrated systems. I specialize in low-level system design, interrupt-driven firmware architecture, real-time embedded systems, and deterministic embedded design.

I have strong foundations in C and Embedded C, microcontroller-based system development, and memory-efficient firmware engineering. My work focuses on building modular, scalable, and event-driven embedded architectures using platforms such as MSP430, ESP32, and ARM-based systems.

Core Technical Strengths:

  • Interrupt Service Routine (ISR) based firmware design
  • RTOS task scheduling and inter-task communication
  • Hardware–software integration and peripheral control
  • Memory optimization and stack management
  • GPIO, Timers, UART, SPI, I2C interfacing
  • State machine and modular architecture design

Alongside firmware development, I actively explore AI, Computer Vision, and Edge AI optimization. I build CNN-based models and deploy lightweight ML inference systems on embedded platforms using TinyML and TensorFlow Lite.

My projects range from autonomous robots and interrupt-driven event systems to arbitrary precision computation engines, LSB-based steganography tools, and AI-powered waste classification systems.
